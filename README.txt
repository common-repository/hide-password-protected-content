Hide password protected content from displaying in recent posts on the front end of your website.

== Description ==

This plugin will prevent your password protected content from displaying on your blog, in the recent posts widget or anywhere else on the front end of your website that uses the dynamic WordPress loop.  Password protected content will still be accessible through a direct link.

Learn more about me on my <a href="https://scottwyden.com/" target="_blank"> photo website</a>.

== Installation ==

Simply install and sit back.  No configuration required.

1. Upload `hide-password-protected-content.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
